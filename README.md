# Java Swing Login System

## Description
This project is a simple Java Swing Login System developed using Apache NetBeans IDE. It provides a graphical user interface (GUI) where users can enter a username and password. The system validates the credentials and displays a success or error message.

## Features
- User-friendly login interface
- Username and password fields
- Login button to validate credentials
- Reset button to clear input fields
- Success and error message dialogs

## Technologies Used
- Java
- Java Swing
- Apache NetBeans IDE

## How to Run
1. Open the project in Apache NetBeans.
2. Build the project.
3. Run the application.
4. Enter the correct username and password.
5. Click the **Login** button.

## Expected Output
- If the correct username and password are entered, the system displays:
  ```
  Login Successful
  ```
- If incorrect credentials are entered, the system displays:
  ```
  Incorrect username or password
  ```
